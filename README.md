## 💡 What Is This?

A Flutter Web UI concept for a university campus super-app. No backend, no auth, just a fully designed, interactive 5-screen UI build exploring what a modern student platform could look like.

Think GPA tracking, campus news, a student marketplace, and profile management; all in one clean interface.

---

## 📱 Screens

| Screen | Description |
|--------|-------------|
| 🔐 **Login** | University email login with Remember Me |
| 🏠 **Home** | Personalized dashboard; GPA card, next class, campus news feed |
| 📰 **News Feed** | Featured articles + categorized campus updates |
| 🛒 **Marketplace** | Buy & sell between students; textbooks, laptops, accessories |
| 👤 **Profile** | Student info, my listings, stats & profile strength tracker |

---

## ✨ Highlights

- 🎨 Clean navy & white design system throughout
- 🛒 Marketplace with category filters (All, Textbooks, Laptops, Accessories)
- 📊 Profile strength indicator with completion prompts
- ✅ Verified Student badge system
- 📰 Featured news banner with article previews
- 🔔 Notification bell & search in header
